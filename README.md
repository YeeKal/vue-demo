## command tool

**project setup**

```bash
# create project
npm update -g @vue/cli
vue create <PROJECT-NAME>

# serve
yarn serve

# build
yarn build

#  yarn command
yarn init  # 初始化项目
yarn install    # 安装依赖
yarn add package-name #add new dependency
```

**use tailwildcss**

```bash
# Using Yarn
yarn add -D tailwindcss
npx tailwindcss init -p
'''
